Overview

This course introduces the logic and processes behind descriptive and inferential statistics in psychology. Topics include frequencies, measures of central tendency and variability, statistical inference, significance testing, t-tests, ANOVAs, correlation, regression, chi-square tests, and data analysis using statistical software.
